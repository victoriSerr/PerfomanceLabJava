task3
аргументы командной строки подаютс с помощью колючей
-log (путь к лог файлу)
-from (начало периода)
-to (конец периода)

Пример команды
java -jar task3.jar -log test.log -from 2021-01-03T19:11:01.890 -to 2022-01-03T23:11:01.881

результатом программы является файл result.csv, который появися в папке и исполняемым модулем
